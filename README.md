# CPS-688-Algorithm-Implementations
The following algorithms are implemented the given Intellij Java Projects:

1. Breadth First Search
2. Depth First Search
3. Longest Increasing Subsequence (implemented using Dynamic Programming)
4. Minimum Spanning Tree
5. Knapsack Problem (implemented using Dynamic Programming)
6. Bipartite Matching
7. Boyer Moore String search algorithm
8. Rabin Karp String Search algorithm using a deterministic finite automata
